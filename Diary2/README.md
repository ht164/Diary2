# Diary2

Very simple diary system using node.js.

## Usage

### boot

$ node bin/www

### show diary site

access http://hostname/ .
write diary in markdown.

### post diary

access http://hostname/html/post.html .

## Using libraries.

 * Express
 * Mongoose
 * UnderscoreJS
 * node-rss
 * jQuery
 * jQuery.inview
 * Bootstrap
 * RequireJS
 * ... and so on ...
